# Quiz Project

## Project Overview
This is a simple Java quiz application that allows users to answer questions loaded from a text file. The program tracks scores based on the user's responses.

## Features
- Console-based quiz interface.
- Questions loaded from `quiz.txt`.
- Score tracking with feedback at the end of the quiz.
